Comment activer la géolocalisation

1) Un message devrait déjà être affiché vous demandant d'autoriser la géolocalisation.

2) Si rien ne s'affiche, essayez d'actualiser la page, sinon effacez le cache de votre navigateur puis rechargez la page web.

3) Si ensuite vous voyez le message, cliquez sur autoriser et l'application devrait fonctionner correctement.

Comment utiliser l'application

- Pour visualiser les détails d'un restaurant, vous devez soit cliquer sur son élément dans la liste ou sur son marqueur.

- Lorsque vous consultez les détails d'un restaurant, vous pouvez également y ajouter votre propre avis.

- Vous pouvez ajouter votre propre restaurant en cliquant sur un endroit spécifique de la carte, vous devrez ensuite remplir un formulaire avec ses détails afin de pouvoir l'ajouter correctement avec les autres restaurants.

- Si vous voulez filtrer les restaurants, il y a un petit outil sous la carte que vous pouvez utiliser pour cela.

- En naviguant sur la carte, vous remarquerez l'apparition d'un petit bouton de recherche à droite de celle-ci, vous pourrez l'utiliser pour rechercher d'autres restaurants en dehors de votre position actuelle.
